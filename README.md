# flutter_vonage_video

## Install

### Android

- Update `android/app/build.gradle`

```
android {
  ...
  compileOptions {
    sourceCompatibility JavaVersion.VERSION_1_8
    targetCompatibility JavaVersion.VERSION_1_8
  }
  ...
}

dependencies {
  ...
  implementation 'com.opentok.android:opentok-android-sdk:2.18.1'
  ...
}
```
